# SafeBid – Verified Distributed Auction Marketplace

Mono-repo for our Software Engineering project.

## Structure

- `auction_service/` – Django auction platform (web app, RPyC client).
- `recommendation_service/` – Python microservice with RPyC server.
- `protocols/` – Scribble protocols and projections.
- `clients/` – Scripts simulating Buyer/Seller.
- `docs/` – Architecture and design documentation.
- `scripts/` – Helper scripts for running everything.
